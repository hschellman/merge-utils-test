usage: merge [-h] [-c CONFIG] [-v] FUNCTION ...

Command line interface for merge_utils

positional arguments:
  FUNCTION              function to execute
    dids                list the DIDs of files from MetaCat
    paths               list the paths of files from a list
    pfns                list the PFNs of files from Rucio
    merge               actualy merge files

optional arguments:
  -h, --help            show this help message and exit
  -c CONFIG, --config CONFIG
                        a configuration file
  -v, --verbose         print more verbose output
