metacat_utils¶

Utility functions for interacting with the MetaCat web API.

metacat_utils.find_logical_files(query: str | None = None, filelist: list | None = None, config: dict | None = None) → DataSet[source]¶

Retrieve logical file information from MetaCat based on an MQL query or a list of DIDs. Returns a DataSet of unique files if the metadata is consistent, otherwise an empty DataSet. The config dictionary can be used to set the following options: - allow_missing: allow missing files - allow_duplicates: allow duplicate files - checked_fields: list of metadata fields to check for consistency

metacat_utils.log_bad_files(files: dict, msg: str) → int[source]¶

Log a message for missing or duplicate files
